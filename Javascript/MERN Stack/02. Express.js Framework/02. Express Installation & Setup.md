# 02. Express Installation & Setup

## Prerequisites

- Node.js installed (v14+ recommended)
- npm (comes with Node.js)

**Check versions:**

```bash
node --version
npm --version
```

---

## Installation Steps

### 1. Create Project Folder

```bash
mkdir my-express-app
cd my-express-app
```

### 2. Initialize npm

```bash
npm init -y
```

This creates `package.json`

### 3. Install Express

```bash
npm install express
```

---

## Project Structure

```
my-express-app/
├── node_modules/
├── package.json
├── package-lock.json
└── app.js
```

---

## Verify Installation

**Create `app.js`:**

```javascript
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Express is working!');
});

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

**Run:**

```bash
node app.js
```

**Visit:** `http://localhost:3000`

---

## Install nodemon (Optional)

Auto-restart server on file changes:

```bash
npm install --save-dev nodemon
```

**Update `package.json`:**

```json
"scripts": {
  "start": "node app.js",
  "dev": "nodemon app.js"
}
```

**Run with nodemon:**

```bash
npm run dev
```

---

## Quick Setup (One Command)

```bash
mkdir my-app && cd my-app && npm init -y && npm install express
```
