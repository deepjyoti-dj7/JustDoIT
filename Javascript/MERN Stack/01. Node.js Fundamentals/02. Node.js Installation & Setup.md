# 02. Node.js Installation & Setup

## Installation Methods

### 1. **Official Installer** (Recommended for beginners)

**Windows & macOS:**
- Visit [nodejs.org](https://nodejs.org)
- Download LTS (Long Term Support) version
- Run installer and follow instructions

**Linux (Ubuntu/Debian):**
```bash
# Using NodeSource
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### 2. **NVM (Node Version Manager)** (Recommended for developers)

**Install NVM:**
```bash
# macOS/Linux
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Verify installation
nvm --version
```

**Install Node.js with NVM:**
```bash
# Install latest LTS
nvm install --lts

# Install specific version
nvm install 18.17.0

# List installed versions
nvm list

# Switch version
nvm use 18.17.0

# Set default version
nvm alias default 18.17.0
```

## Verify Installation

```bash
# Check Node.js version
node --version
# or
node -v

# Check NPM version
npm --version
# or
npm -v

# Check installation path
which node
```

## Your First Node.js Program

### 1. **Create a file** `hello.js`

```javascript
console.log('Hello, Node.js!');
console.log('Node version:', process.version);
console.log('Platform:', process.platform);
```

### 2. **Run the program**

```bash
node hello.js
```

**Output:**
```
Hello, Node.js!
Node version: v18.17.0
Platform: darwin
```

## Node.js REPL

**REPL** = Read-Eval-Print-Loop (Interactive shell)

```bash
# Start REPL
node

# Try commands
> 2 + 2
4
> const name = 'Node.js'
undefined
> name
'Node.js'
> console.log('Hello')
Hello
undefined

# Exit REPL
> .exit
# or Ctrl+C twice
```

## Initialize a New Project

```bash
# Create project directory
mkdir my-node-app
cd my-node-app

# Initialize npm (creates package.json)
npm init -y

# Install dependencies
npm install express

# Create entry file
touch index.js
```

## Global vs Local Installation

### **Local Installation** (Default)
```bash
# Installs in node_modules/
npm install express
```

### **Global Installation**
```bash
# Available system-wide
npm install -g nodemon

# Use globally installed package
nodemon app.js
```

**When to use global?**
- CLI tools (nodemon, create-react-app)
- Utilities used across projects

## Useful NPM Commands

```bash
# Initialize project
npm init
npm init -y  # Skip questions

# Install package
npm install <package-name>
npm i <package-name>  # Short form

# Install as dev dependency
npm install --save-dev <package-name>
npm i -D <package-name>

# Install specific version
npm install express@4.18.0

# Uninstall package
npm uninstall <package-name>

# Update packages
npm update

# List installed packages
npm list
npm list --depth=0  # Top level only

# Check outdated packages
npm outdated

# Run scripts
npm run <script-name>
npm start
npm test
```

## Troubleshooting

### Issue: `command not found: node`

**Solution:**
```bash
# Check if installed
which node

# Add to PATH (macOS/Linux)
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### Issue: Permission errors on npm install

**Solution:**
```bash
# Fix npm permissions (macOS/Linux)
sudo chown -R $USER /usr/local/lib/node_modules
```
