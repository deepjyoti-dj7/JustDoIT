# Object Basics

## Creating Objects

### Object Literal
```javascript
const person = {
  name: 'John',
  age: 30,
  city: 'New York'
};

// Empty object
const empty = {};

// Nested objects
const user = {
  name: 'Jane',
  address: {
    street: '123 Main St',
    city: 'Boston'
  }
};
```

### Object Constructor
```javascript
const obj1 = new Object();
obj1.name = 'John';

// From null prototype
const obj2 = Object.create(null);  // No inherited properties

// With prototype
const proto = {greet: function() { return 'Hello'; }};
const obj3 = Object.create(proto);
console.log(obj3.greet());  // 'Hello'
```

### Object.fromEntries()
```javascript
// From array of entries
const entries = [['name', 'John'], ['age', 30]];
const obj = Object.fromEntries(entries);
console.log(obj);  // {name: 'John', age: 30}

// From Map
const map = new Map([['a', 1], ['b', 2]]);
const fromMap = Object.fromEntries(map);
console.log(fromMap);  // {a: 1, b: 2}
```

## Property Access

### Dot Notation
```javascript
const person = {name: 'John', age: 30};

console.log(person.name);  // 'John'
console.log(person.age);   // 30
console.log(person.city);  // undefined
```

### Bracket Notation
```javascript
const person = {name: 'John', age: 30};

console.log(person['name']);  // 'John'

// Dynamic property access
const key = 'age';
console.log(person[key]);     // 30

// Properties with special characters
const obj = {'first-name': 'John'};
console.log(obj['first-name']);
```

### Optional Chaining
```javascript
const user = {
  name: 'John',
  address: {
    city: 'NYC'
  }
};

console.log(user.address?.city);     // 'NYC'
console.log(user.contact?.phone);    // undefined (no error)
console.log(user.getName?.());       // undefined (safe method call)
```

## Adding/Deleting Properties

### Adding Properties
```javascript
const person = {name: 'John'};

// Dot notation
person.age = 30;

// Bracket notation
person['city'] = 'NYC';

// Multiple properties
Object.assign(person, {
  email: 'john@example.com',
  phone: '123-456'
});

console.log(person);
// {name: 'John', age: 30, city: 'NYC', email: 'john@example.com', phone: '123-456'}
```

### Deleting Properties
```javascript
const person = {name: 'John', age: 30, city: 'NYC'};

delete person.city;
console.log(person);  // {name: 'John', age: 30}

// Check if property exists
console.log('city' in person);  // false
console.log(person.hasOwnProperty('name'));  // true
```

### Property Existence
```javascript
const obj = {a: 1, b: undefined};

// in operator (checks own + inherited)
console.log('a' in obj);              // true
console.log('toString' in obj);       // true (inherited)

// hasOwnProperty (own only)
console.log(obj.hasOwnProperty('a')); // true
console.log(obj.hasOwnProperty('toString'));  // false

// Object.hasOwn() (ES2022, safer)
console.log(Object.hasOwn(obj, 'a')); // true

// undefined check
console.log(obj.a !== undefined);     // true
console.log(obj.b !== undefined);     // false (b exists but is undefined)
```

## Property Descriptors

### Get Descriptor
```javascript
const obj = {name: 'John'};

const descriptor = Object.getOwnPropertyDescriptor(obj, 'name');
console.log(descriptor);
// {
//   value: 'John',
//   writable: true,
//   enumerable: true,
//   configurable: true
// }
```

### Define Property
```javascript
const obj = {};

Object.defineProperty(obj, 'name', {
  value: 'John',
  writable: false,      // Cannot be changed
  enumerable: true,     // Shows in for...in
  configurable: false   // Cannot be deleted or reconfigured
});

obj.name = 'Jane';      // Silently fails (strict mode throws error)
console.log(obj.name);  // 'John'

delete obj.name;        // Fails
console.log(obj.name);  // 'John'
```

### Getters and Setters
```javascript
const person = {
  firstName: 'John',
  lastName: 'Doe',
  
  get fullName() {
    return `${this.firstName} ${this.lastName}`;
  },
  
  set fullName(name) {
    [this.firstName, this.lastName] = name.split(' ');
  }
};

console.log(person.fullName);  // 'John Doe'
person.fullName = 'Jane Smith';
console.log(person.firstName);  // 'Jane'
```

## Object Destructuring

### Basic Destructuring
```javascript
const person = {name: 'John', age: 30, city: 'NYC'};

const {name, age} = person;
console.log(name, age);  // 'John' 30

// Rename variables
const {name: personName, age: personAge} = person;
console.log(personName);  // 'John'
```

### Default Values
```javascript
const person = {name: 'John'};

const {name, age = 25, city = 'Unknown'} = person;
console.log(age, city);  // 25 'Unknown'
```

### Nested Destructuring
```javascript
const user = {
  name: 'John',
  address: {
    city: 'NYC',
    zip: '10001'
  }
};

const {name, address: {city, zip}} = user;
console.log(city, zip);  // 'NYC' '10001'
```

### Rest in Objects
```javascript
const person = {name: 'John', age: 30, city: 'NYC', country: 'USA'};

const {name, ...rest} = person;
console.log(name);  // 'John'
console.log(rest);  // {age: 30, city: 'NYC', country: 'USA'}
```

### Function Parameters
```javascript
function greet({name, age = 18}) {
  console.log(`Hello ${name}, age ${age}`);
}

greet({name: 'John', age: 30});  // Hello John, age 30
greet({name: 'Jane'});            // Hello Jane, age 18
```

## Computed Property Names

### Dynamic Keys
```javascript
const key = 'name';
const value = 'John';

const obj = {
  [key]: value,
  [`${key}Upper`]: value.toUpperCase()
};

console.log(obj);  // {name: 'John', nameUpper: 'JOHN'}
```

### Expression as Key
```javascript
const prefix = 'user';

const obj = {
  [prefix + 'Name']: 'John',
  [prefix + 'Age']: 30,
  ['get' + prefix.charAt(0).toUpperCase() + prefix.slice(1)]() {
    return this.userName;
  }
};

console.log(obj.userName);     // 'John'
console.log(obj.getUser());    // 'John'
```

### Symbol Keys
```javascript
const id = Symbol('id');
const user = {
  name: 'John',
  [id]: 12345
};

console.log(user[id]);  // 12345

// Symbols are not enumerable
for (let key in user) {
  console.log(key);  // Only 'name', not Symbol(id)
}
```

## Shorthand Syntax

### Property Shorthand
```javascript
const name = 'John';
const age = 30;

// Old way
const person1 = {name: name, age: age};

// Shorthand
const person2 = {name, age};

console.log(person2);  // {name: 'John', age: 30}
```

### Method Shorthand
```javascript
const obj = {
  // Old way
  greet1: function() {
    return 'Hello';
  },
  
  // Shorthand
  greet2() {
    return 'Hello';
  }
};

console.log(obj.greet2());  // 'Hello'
```

## Key Takeaways

- Objects are **key-value pairs**, keys are strings or Symbols
- Use **dot notation** for known properties, **brackets** for dynamic access
- **Optional chaining** `?.` prevents errors when accessing nested properties
- **delete** operator removes properties
- Property **descriptors** control writability, enumerability, configurability
- **Getters/setters** provide computed properties
- **Destructuring** extracts properties into variables
- **Computed property names** `[expression]` for dynamic keys
- **Shorthand syntax** for cleaner object literals
- Use **Object.hasOwn()** instead of `hasOwnProperty()` (ES2022)
