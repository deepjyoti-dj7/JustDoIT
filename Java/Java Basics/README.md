- Java basics (JDK, JRE, JVM)
- Data types, variables, operators

- Arrays and Strings
- Basic I/O operations

**Practice**: Write simple programs, solve basic problems

---

**Goal**: Master object-oriented programming

- Classes, objects, constructors
- Inheritance and polymorphism
- Encapsulation and abstraction
- Interfaces and abstract classes
- Static, final, inner classes

**Practice**: Design class hierarchies, implement real-world scenarios

---

### Phase 3: Exception Handling (Week 5)
**Goal**: Handle errors gracefully

- Exception hierarchy
- try-catch-finally blocks
- Custom exceptions
- Best practices

**Practice**: Write robust code with proper exception handling

---

### Phase 4: Generics & Collections (Week 6)
**Goal**: Use type-safe collections

- Generic classes and methods
- Wildcards and bounded types
- Collection framework basics

**Practice**: Implement generic data structures

---

### Phase 5: Modern Java (Week 7-8)
**Goal**: Write modern, functional code

- Lambda expressions
- Functional interfaces
- Stream API
- Optional class
- Method references

**Practice**: Refactor imperative code to functional style

---

### Phase 6: Advanced Topics (Week 9-10)
**Goal**: Deep dive into advanced features

- File I/O and NIO

**Practice**: Build real-world applications

### Phase 7: Java 8+ Features (Week 11)

- New features in Java 9-17

**Practice**: Migrate legacy code to modern Java

---

### Phase 8: Interview Preparation (Week 12)
**Goal**: Master interview questions

- Core Java concepts review
