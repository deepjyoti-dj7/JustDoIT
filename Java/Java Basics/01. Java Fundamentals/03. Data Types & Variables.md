# üìä Data Types & Variables

## üìñ Overview

Java is a **strongly-typed** language, meaning every variable must have a declared type.

## üî∑ Primitive Data Types

Java has **8 primitive data types**:

| Type | Size | Range | Default | Wrapper Class |
|------|------|-------|---------|---------------|
| `byte` | 8 bits | -128 to 127 | 0 | Byte |
| `short` | 16 bits | -32,768 to 32,767 | 0 | Short |
| `int` | 32 bits | -2¬≥¬π to 2¬≥¬π-1 | 0 | Integer |
| `long` | 64 bits | -2‚Å∂¬≥ to 2‚Å∂¬≥-1 | 0L | Long |
| `float` | 32 bits | ~¬±3.4E38 (6-7 decimal places) | 0.0f | Float |
| `double` | 64 bits | ~¬±1.8E308 (15 decimal places) | 0.0d | Double |
| `char` | 16 bits | 0 to 65,535 (Unicode) | '\u0000' | Character |
| `boolean` | 1 bit | true or false | false | Boolean |

## üíª Example 4: Type Casting

```java
public class TypeCastingDemo {
    public static void main(String[] args) {
        // Widening (Implicit) - smaller to larger
        System.out.println("=== Widening (Automatic) ===");
        byte b = 10;
        int i = b;           // byte ‚Üí int
        long l = i;          // int ‚Üí long
        float f = l;         // long ‚Üí float
        double d = f;        // float ‚Üí double
        
        System.out.println("byte: " + b);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
        System.out.println("float: " + f);
        System.out.println("double: " + d);
        
        // Narrowing (Explicit) - larger to smaller
        System.out.println("\n=== Narrowing (Manual) ===");
        double d2 = 99.99;
        int i2 = (int) d2;          // Truncates decimal part
        short s = (short) i2;
        byte b2 = (byte) s;
        
        System.out.println("double: " + d2);
        System.out.println("int: " + i2);
        System.out.println("short: " + s);
        System.out.println("byte: " + b2);
        
        // Data loss in narrowing
        System.out.println("\n=== Data Loss ===");
        int bigValue = 300;
        byte smallValue = (byte) bigValue;  // Overflow
        System.out.println("int 300 ‚Üí byte: " + smallValue);  // -44 (overflow)
        
        // Character casting
        System.out.println("\n=== Character Casting ===");
        char ch = 'A';
        int ascii = ch;              // char ‚Üí int (implicit)
        System.out.println("'A' ‚Üí int: " + ascii);  // 65
        
        int num = 66;
        char character = (char) num;  // int ‚Üí char (explicit)
        System.out.println("66 ‚Üí char: " + character);  // 'B'
    }
}
```

## üíª Example 5: Wrapper Classes

```java
public class WrapperClassDemo {
    public static void main(String[] args) {
        // Primitive to Wrapper (Boxing)
        int primitiveInt = 10;
        Integer wrapperInt = Integer.valueOf(primitiveInt);  // Explicit
        
        // Autoboxing (Java 5+)
        Integer autoBoxed = primitiveInt;
        
        // Wrapper to Primitive (Unboxing)
        Integer wrapper = 20;
        int primitive = wrapper.intValue();  // Explicit
        
        // Auto-unboxing
        int autoUnboxed = wrapper;
        
        System.out.println("Primitive: " + primitiveInt);
        System.out.println("Wrapper: " + wrapperInt);
        System.out.println("Autoboxed: " + autoBoxed);
        System.out.println("Unboxed: " + primitive);
        System.out.println("Auto-unboxed: " + autoUnboxed);
        
        // Wrapper class utility methods
        System.out.println("\n=== Wrapper Utilities ===");
        System.out.println("MAX_VALUE: " + Integer.MAX_VALUE);
        System.out.println("MIN_VALUE: " + Integer.MIN_VALUE);
        System.out.println("SIZE: " + Integer.SIZE + " bits");
        System.out.println("BYTES: " + Integer.BYTES + " bytes");
        
        // String conversion
        String str = "123";
        int num = Integer.parseInt(str);
        String numStr = Integer.toString(456);
        
        System.out.println("\nString to int: " + num);
        System.out.println("int to String: " + numStr);
        
        // Comparison
        Integer a = 100;
        Integer b = 100;
        Integer c = 200;
        
        System.out.println("\na == b: " + (a == b));      // true (cache)
        System.out.println("a.equals(b): " + a.equals(b));  // true
        System.out.println("a.compareTo(c): " + a.compareTo(c));  // -1
    }
}
```

public class DefaultValuesDemo {

## üíª Example 7: Variable Scope

```java
public class VariableScopeDemo {
    // Class-level variable (accessible everywhere in class)
    static int classVar = 100;
    
    // Instance variable (accessible to all instance methods)
    int instanceVar = 200;
    
    public void method1() {
        // Local variable (scope: within this method only)
        int localVar = 300;
        
        System.out.println("In method1:");
        System.out.println("  Class variable: " + classVar);
        System.out.println("  Instance variable: " + instanceVar);
        System.out.println("  Local variable: " + localVar);
        
        // Block scope
        if (true) {
            int blockVar = 400;
            System.out.println("  Block variable: " + blockVar);
        }
        // System.out.println(blockVar);  // Error: out of scope
    }
    
    public void method2() {
        // localVar from method1 is not accessible here
        // System.out.println(localVar);  // Error!
        
        System.out.println("\nIn method2:");
        System.out.println("  Class variable: " + classVar);
        System.out.println("  Instance variable: " + instanceVar);
    }
    
    public static void main(String[] args) {
        VariableScopeDemo demo = new VariableScopeDemo();
        demo.method1();
        demo.method2();
    }
}
```

## üìä Type Promotion Rules

```java
public class TypePromotionDemo {
    public static void main(String[] args) {
        byte b = 10;
        short s = 20;
        int i = 30;
        long l = 40L;
        float f = 50.5f;
        double d = 60.6;
        
        // byte + short ‚Üí int
        int result1 = b + s;
        System.out.println("byte + short ‚Üí int: " + result1);
        
        // int + long ‚Üí long
        long result2 = i + l;
        System.out.println("int + long ‚Üí long: " + result2);
        
        // long + float ‚Üí float
        float result3 = l + f;
        System.out.println("long + float ‚Üí float: " + result3);
        
        // float + double ‚Üí double
        double result4 = f + d;
        System.out.println("float + double ‚Üí double: " + result4);
        
        // char promoted to int
        char ch = 'A';
        int ascii = ch + 1;
        System.out.println("'A' + 1 = " + ascii + " (" + (char)ascii + ")");
    }
}
```

## üí° Best Practices

1. ‚úÖ Use appropriate data types (don't use `long` when `int` suffices)
2. ‚úÖ Use `double` for decimal calculations (more precision than `float`)
3. ‚úÖ Use constants (`final`) for unchanging values
4. ‚úÖ Follow naming conventions (camelCase for variables, UPPER_CASE for constants)
5. ‚úÖ Initialize variables before use (especially local variables)
6. ‚úÖ Use wrapper classes when null values are needed
7. ‚úÖ Be careful with narrowing conversions (data loss)
8. ‚úÖ Use underscore in large numeric literals for readability

## ‚ö†Ô∏è Common Pitfalls

```java
public class CommonPitfalls {
    public static void main(String[] args) {
        // 1. Integer division
        int a = 5 / 2;           // 2 (not 2.5)
        double b = 5 / 2;        // 2.0 (not 2.5)
        double c = 5.0 / 2;      // 2.5 (correct)
        
        System.out.println("5 / 2 (int): " + a);
        System.out.println("5 / 2 ‚Üí double: " + b);
        System.out.println("5.0 / 2: " + c);
        
        // 2. Overflow
        int max = Integer.MAX_VALUE;
        int overflow = max + 1;  // Wraps to MIN_VALUE
        System.out.println("\nMax + 1: " + overflow);
        
        // 3. Floating-point precision
        double x = 0.1 + 0.2;
        System.out.println("\n0.1 + 0.2 = " + x);  // Not exactly 0.3!
        
        // 4. Uninitialized local variable
        // int local;
        // System.out.println(local);  // Error!
    }
}
```
