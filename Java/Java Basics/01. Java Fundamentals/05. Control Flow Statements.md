```java
public class SwitchStatementDemo {
    public static void main(String[] args) {
        // Day of week
        int day = 3;
        String dayName;
        
        switch (day) {
            case 1:
                dayName = "Monday";
                break;
            case 2:
                dayName = "Tuesday";
                break;
            case 3:
                dayName = "Wednesday";
                break;
            case 4:
                dayName = "Thursday";
                break;
            case 5:
                dayName = "Friday";
                break;
            case 6:
                dayName = "Saturday";
                break;
            case 7:
                dayName = "Sunday";
                break;
            default:
                dayName = "Invalid day";
        }
        
        System.out.println("Day " + day + " is " + dayName);
        
        // Fall-through behavior (no break)
        int month = 2;
        int year = 2024;
        int days;
        
        switch (month) {
            case 1: case 3: case 5: case 7: case 8: case 10: case 12:
                days = 31;
                break;
            case 4: case 6: case 9: case 11:
                days = 30;
                break;
            case 2:
                days = (year % 4 == 0) ? 29 : 28;
                break;
            default:
                days = 0;
        }
        
        System.out.println("Month " + month + " has " + days + " days");
    }
}
```

## ðŸ’» Example 5: switch Expression (Java 12+)

```java
public class SwitchExpressionDemo {
    public static void main(String[] args) {
        // Switch expression with arrow syntax
        int day = 3;
        
        String dayType = switch (day) {
            case 1, 2, 3, 4, 5 -> "Weekday";
            case 6, 7 -> "Weekend";
            default -> "Invalid day";
        };
        
        System.out.println("Day " + day + " is a " + dayType);
        
        // Switch expression with yield
        int month = 2;
        
        String season = switch (month) {
            case 12, 1, 2 -> {
                System.out.println("Cold season");
                yield "Winter";
            }
            case 3, 4, 5 -> {
                System.out.println("Pleasant season");
                yield "Spring";
            }
            case 6, 7, 8 -> {
                System.out.println("Hot season");
                yield "Summer";
            }
            case 9, 10, 11 -> {
                System.out.println("Moderate season");
                yield "Autumn";
            }
            default -> "Invalid month";
        };
        
        System.out.println("Month " + month + " is in " + season);
        
        // Switch with enum
        Day today = Day.WEDNESDAY;
        
        int numLetters = switch (today) {
            case MONDAY, FRIDAY, SUNDAY -> 6;
            case TUESDAY -> 7;
            case THURSDAY, SATURDAY -> 8;
            case WEDNESDAY -> 9;
        };
        
        System.out.println(today + " has " + numLetters + " letters");
    }
    
    enum Day {
        MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY
    }
}
```

```java
        // Check even/odd
        int num = 17;
        System.out.println(num + " is " + ((num % 2 == 0) ? "even" : "odd"));
```

## ðŸ’» Example 7: break Statement

```java
public class BreakStatementDemo {
    public static void main(String[] args) {
        // break in loop
        System.out.println("=== break in for loop ===");
        for (int i = 1; i <= 10; i++) {
            if (i == 6) {
                break;  // Exit loop when i is 6
            }
            System.out.print(i + " ");
        }
        
        // break in while loop
        System.out.println("\n\n=== break in while loop ===");
        int count = 1;
        while (true) {
            System.out.print(count + " ");
            if (count == 5) {
                break;  // Exit infinite loop
            }
            count++;
        }
        
        // break in nested loop (only exits inner loop)
        System.out.println("\n\n=== break in nested loop ===");
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 5; j++) {
                if (j == 3) {
                    break;  // Exits only inner loop
                }
                System.out.print(i + "-" + j + " ");
            }
            System.out.println();
        }
        
        // Labeled break (exits outer loop)
        System.out.println("\n=== Labeled break ===");
        outer: for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 5; j++) {
                if (j == 3) {
                    break outer;  // Exits outer loop
                }
                System.out.print(i + "-" + j + " ");
            }
            System.out.println();
        }
    }
}
```

## ðŸ’» Example 8: continue Statement

```java
public class ContinueStatementDemo {
    public static void main(String[] args) {
        // continue in for loop - skip even numbers
        System.out.println("=== Print odd numbers ===");
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue;  // Skip even numbers
            }
            System.out.print(i + " ");
        }
        
        // continue in while loop
        System.out.println("\n\n=== Skip multiples of 3 ===");
        int num = 0;
        while (num < 15) {
            num++;
            if (num % 3 == 0) {
                continue;  // Skip multiples of 3
            }
            System.out.print(num + " ");
        }
        
        // continue in nested loop
        System.out.println("\n\n=== continue in nested loop ===");
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 5; j++) {
                if (j == 3) {
                    continue;  // Skip j=3 in inner loop
                }
                System.out.print(i + "-" + j + " ");
            }
            System.out.println();
        }
        
        // Labeled continue
        System.out.println("\n=== Labeled continue ===");
        outer: for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 5; j++) {
                if (j == 3) {
                    continue outer;  // Continue outer loop
                }
                System.out.print(i + "-" + j + " ");
            }
            System.out.println();
        }
    }
}
```

## ðŸ’» Example 9: return Statement

```java
public class ReturnStatementDemo {
    public static void main(String[] args) {
        // Method with return value
        int sum = add(10, 20);
        System.out.println("Sum: " + sum);
        
        // Method with early return
        String result = checkEligibility(25, true);
        System.out.println(result);
        
        // Method with multiple return points
        String grade = getGrade(85);
        System.out.println("Grade: " + grade);
        
        // Void method with return
        printMessage(5);
    }
    
    // Return with value
    static int add(int a, int b) {
        return a + b;
    }
    
    // Early return
    static String checkEligibility(int age, boolean hasLicense) {
        if (age < 18) {
            return "Not eligible - Too young";
        }
        if (!hasLicense) {
            return "Not eligible - No license";
        }
        return "Eligible";
    }
    
    // Multiple return points
    static String getGrade(int marks) {
        if (marks >= 90) return "A";
        if (marks >= 80) return "B";
        if (marks >= 70) return "C";
        if (marks >= 60) return "D";
        return "F";
    }
    
    // Return in void method
    static void printMessage(int count) {
        if (count <= 0) {
            return;  // Early exit
        }
        for (int i = 1; i <= count; i++) {
            System.out.println("Message " + i);
        }
    }
}
```

## ðŸ’» Example 10: Practical Control Flow Examples

```java
import java.util.Scanner;

public class PracticalControlFlowDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Menu-driven program
        System.out.println("=== Calculator ===");
        System.out.println("1. Add");
        System.out.println("2. Subtract");
        System.out.println("3. Multiply");
        System.out.println("4. Divide");
        System.out.print("Enter choice: ");
        int choice = scanner.nextInt();
        
        System.out.print("Enter first number: ");
        double num1 = scanner.nextDouble();
        System.out.print("Enter second number: ");
        double num2 = scanner.nextDouble();
        
        double result = switch (choice) {
            case 1 -> num1 + num2;
            case 2 -> num1 - num2;
            case 3 -> num1 * num2;
            case 4 -> {
                if (num2 == 0) {
                    System.out.println("Error: Division by zero");
                    yield 0;
                }
                yield num1 / num2;
            }
            default -> {
                System.out.println("Invalid choice");
                yield 0;
            }
        };
        
        System.out.println("Result: " + result);
        
        // Password validation
        System.out.print("\nEnter password: ");
        String password = scanner.next();
        
        if (validatePassword(password)) {
            System.out.println("Password is valid");
        } else {
            System.out.println("Password is invalid");
        }
        
        scanner.close();
    }
    
    static boolean validatePassword(String password) {
        if (password.length() < 8) {
            System.out.println("Password must be at least 8 characters");
            return false;
        }
        
        boolean hasUpper = false, hasLower = false, hasDigit = false;
        
        for (char ch : password.toCharArray()) {
            if (Character.isUpperCase(ch)) hasUpper = true;
            if (Character.isLowerCase(ch)) hasLower = true;
            if (Character.isDigit(ch)) hasDigit = true;
        }
        
        if (!hasUpper) {
            System.out.println("Password must contain uppercase letter");
            return false;
        }
        if (!hasLower) {
            System.out.println("Password must contain lowercase letter");
            return false;
        }
        if (!hasDigit) {
            System.out.println("Password must contain digit");
            return false;
        }
        
        return true;
    }
}
```

## ðŸ“Š Control Flow Statement Comparison

| Statement | Use Case | Can Return Value | Breaks Execution |
|-----------|----------|------------------|------------------|
| `if` | Single condition | No | No |
| `if-else` | Two alternatives | No | No |
| `if-else-if` | Multiple conditions | No | No |
| `switch` | Multiple values | Yes (Java 12+) | Yes (with break) |
| `ternary` | Simple condition | Yes | No |
| `break` | Exit loop/switch | No | Yes |
| `continue` | Skip iteration | No | No |
| `return` | Exit method | Yes (except void) | Yes |

## ðŸ’¡ Best Practices

1. âœ… Use `switch` for multiple equality checks
2. âœ… Use `if-else` for range/complex conditions
3. âœ… Always use braces `{}` for if/else blocks
4. âœ… Avoid deeply nested if statements (use methods)
5. âœ… Use switch expressions (Java 12+) for cleaner code
6. âœ… Use early returns to avoid deep nesting
7. âœ… Use labeled break/continue sparingly
8. âœ… Keep ternary operators simple (no deep nesting)

```java
public class ControlFlowPitfalls {
    public static void main(String[] args) {
        // 1. Dangling else problem
        int x = 5;
        if (x > 10)
            if (x > 20)
                System.out.println("x > 20");
        else  // This else belongs to inner if, not outer!
            System.out.println("x <= 20");
        
        // 2. Forgetting break in switch
        int day = 2;
        switch (day) {
            case 1:
                System.out.println("Monday");
                // Missing break - falls through!
            case 2:
                System.out.println("Tuesday");
                break;
        }
        
        // 3. Using == for String comparison
        String s1 = new String("Hello");
        String s2 = new String("Hello");
        if (s1 == s2) {  // Wrong! Compares references
            System.out.println("Equal");
        }
        if (s1.equals(s2)) {  // Correct! Compares content
            System.out.println("Equal");
        }
        
        // 4. Unreachable code
        // if (true) {
        //     return;
        //     System.out.println("This is unreachable");  // Error!
        // }
    }
}
```
