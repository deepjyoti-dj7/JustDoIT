# üèóÔ∏è Classes & Objects

## üìñ Overview

**Classes** and **Objects** are the fundamental building blocks of Object-Oriented Programming (OOP) in Java.

**Key Concepts:**
- **Class**: Blueprint/template for creating objects
- **Object**: Instance of a class (actual entity)
- **Instance Variables**: Data fields that belong to objects
- **Methods**: Behaviors/functions that objects can perform
- **this**: Reference to the current object

---

## üéØ Class vs Object

```
Class (Blueprint):
    - Template/design
    - No memory allocated
    - Defined once
    - Example: Car class
    
Object (Instance):
    - Actual entity created from class
    - Memory allocated
    - Can have multiple instances
    - Example: myCar, yourCar
```

**Analogy:**
- Class = Cookie Cutter (template)
- Object = Cookie (actual product)

## üíª Example 1: Basic Class & Object

```java
// Define a class
class Student {
    // Instance variables (attributes)
    String name;
    int age;
    String course;
    
    // Method (behavior)
    void displayInfo() {
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Course: " + course);
    }
}

public class ClassObjectBasicsDemo {
    public static void main(String[] args) {
        // Create object (instance of Student class)
        Student student1 = new Student();
        
        // Set values to instance variables
        student1.name = "Alice";
        student1.age = 20;
        student1.course = "Computer Science";
        
        // Call method
        System.out.println("=== Student 1 ===");
        student1.displayInfo();
        
        // Create another object
        Student student2 = new Student();
        student2.name = "Bob";
        student2.age = 22;
        student2.course = "Mathematics";
        
        System.out.println("\n=== Student 2 ===");
        student2.displayInfo();
    }
}
```

**Output:**
```
=== Student 1 ===
Name: Alice
Age: 20
Course: Computer Science

=== Student 2 ===
Name: Bob
Age: 22
Course: Mathematics
```

## üíª Example 3: Methods with Parameters

```java
class Calculator {
    int result;
    
    // Method with parameters
    int add(int a, int b) {
        result = a + b;
        return result;
    }
    
    int subtract(int a, int b) {
        result = a - b;
        return result;
    }
    
    int multiply(int a, int b) {
        result = a * b;
        return result;
    }
    
    double divide(int a, int b) {
        if (b != 0) {
            return (double) a / b;
        } else {
            System.out.println("Error: Division by zero!");
            return 0;
        }
    }
    
    void displayResult() {
        System.out.println("Last result: " + result);
    }
}

public class MethodsWithParametersDemo {
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        
        // Use methods
        int sum = calc.add(10, 5);
        System.out.println("10 + 5 = " + sum);
        
        int diff = calc.subtract(10, 5);
        System.out.println("10 - 5 = " + diff);
        
        int product = calc.multiply(10, 5);
        System.out.println("10 * 5 = " + product);
        
        double quotient = calc.divide(10, 5);
        System.out.println("10 / 5 = " + quotient);
        
        calc.displayResult();
        
        // Division by zero
        calc.divide(10, 0);
    }
}
```

## üíª Example 4: The 'this' Keyword

```java
class Employee {
    String name;
    int id;
    double salary;
    
    // Using 'this' to distinguish between parameters and instance variables
    void setDetails(String name, int id, double salary) {
        this.name = name;      // this.name refers to instance variable
        this.id = id;          // name, id, salary refer to parameters
        this.salary = salary;
    }
    
    void displayInfo() {
        System.out.println("ID: " + this.id);
        System.out.println("Name: " + this.name);
        System.out.println("Salary: $" + this.salary);
    }
    
    // Method that uses 'this' to call another method
    void printDetails() {
        System.out.println("Employee Details:");
        this.displayInfo();  // Calling method using 'this'
    }
    
    // Method that returns current object
    Employee getEmployee() {
        return this;  // Returns current object
    }
}

public class ThisKeywordDemo {
    public static void main(String[] args) {
        Employee emp = new Employee();
        
        // Set details
        emp.setDetails("John Doe", 101, 50000);
        
        // Display info
        emp.printDetails();
        
        // Get employee object
        Employee sameEmp = emp.getEmployee();
        System.out.println("\nAre they same? " + (emp == sameEmp));
    }
}
```

**Output:**
```
Employee Details:
ID: 101
Name: John Doe
Salary: $50000.0

Are they same? true
```

## üíª Example 5: Object Reference

```java
class Point {
    int x;
    int y;
    
    void setCoordinates(int x, int y) {
        this.x = x;
        this.y = y;
    }
    
    void display() {
        System.out.println("Point(" + x + ", " + y + ")");
    }
}

public class ObjectReferenceDemo {
    public static void main(String[] args) {
        // Create first object
        Point p1 = new Point();
        p1.setCoordinates(5, 10);
        
        System.out.println("p1: ");
        p1.display();
        
        // Reference assignment (NOT object creation)
        Point p2 = p1;  // p2 refers to same object as p1
        
        System.out.println("p2: ");
        p2.display();
        
        // Modify through p2
        p2.setCoordinates(20, 30);
        
        // Both p1 and p2 show updated values
        System.out.println("\nAfter modifying through p2:");
        System.out.println("p1: ");
        p1.display();
        System.out.println("p2: ");
        p2.display();
        
        // Check if they refer to same object
        System.out.println("\nAre p1 and p2 same object? " + (p1 == p2));
        
        // Create new object
        Point p3 = new Point();
        p3.setCoordinates(5, 10);
        
        System.out.println("\np3: ");
        p3.display();
        
        // Different objects even with same values
        System.out.println("Are p1 and p3 same object? " + (p1 == p3));
    }
}
```

## üíª Example 6: Real-World Example - Bank Account

```java
class BankAccount {
    String accountNumber;
    String accountHolder;
    double balance;
    
    void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
            System.out.println("Deposited: $" + amount);
            System.out.println("New Balance: $" + balance);
        } else {
            System.out.println("Invalid amount!");
        }
    }
    
    void withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            System.out.println("Withdrawn: $" + amount);
            System.out.println("New Balance: $" + balance);
        } else {
            System.out.println("Insufficient balance or invalid amount!");
        }
    }
    
    void checkBalance() {
        System.out.println("Account: " + accountNumber);
        System.out.println("Holder: " + accountHolder);
        System.out.println("Balance: $" + balance);
    }
    
    void transfer(BankAccount targetAccount, double amount) {
        if (amount > 0 && amount <= this.balance) {
            this.withdraw(amount);
            targetAccount.deposit(amount);
            System.out.println("Transfer successful!");
        } else {
            System.out.println("Transfer failed!");
        }
    }
}

public class BankAccountDemo {
    public static void main(String[] args) {
        // Create accounts
        BankAccount account1 = new BankAccount();
        account1.accountNumber = "ACC001";
        account1.accountHolder = "Alice";
        account1.balance = 1000;
        
        BankAccount account2 = new BankAccount();
        account2.accountNumber = "ACC002";
        account2.accountHolder = "Bob";
        account2.balance = 500;
        
        // Display initial balances
        System.out.println("=== Initial State ===");
        account1.checkBalance();
        System.out.println();
        account2.checkBalance();
        
        // Perform operations
        System.out.println("\n=== Deposit to Account 1 ===");
        account1.deposit(500);
        
        System.out.println("\n=== Withdraw from Account 2 ===");
        account2.withdraw(200);
        
        System.out.println("\n=== Transfer from Account 1 to Account 2 ===");
        account1.transfer(account2, 300);
        
        // Display final balances
        System.out.println("\n=== Final State ===");
        account1.checkBalance();
        System.out.println();
        account2.checkBalance();
    }
}
```

## üíª Example 7: Array of Objects

```java
class Book {
    String title;
    String author;
    double price;
    
    void setDetails(String title, String author, double price) {
        this.title = title;
        this.author = author;
        this.price = price;
    }
    
    void display() {
        System.out.println("Title: " + title);
        System.out.println("Author: " + author);
        System.out.println("Price: $" + price);
        System.out.println();
    }
}

public class ArrayOfObjectsDemo {
    public static void main(String[] args) {
        // Create array of objects
        Book[] library = new Book[3];
        
        // Initialize objects
        library[0] = new Book();
        library[0].setDetails("Java Programming", "James Gosling", 45.99);
        
        library[1] = new Book();
        library[1].setDetails("Python Basics", "Guido van Rossum", 39.99);
        
        library[2] = new Book();
        library[2].setDetails("Data Structures", "Robert Sedgewick", 59.99);
        
        // Display all books
        System.out.println("=== Library Books ===");
        for (int i = 0; i < library.length; i++) {
            System.out.println("Book " + (i + 1) + ":");
            library[i].display();
        }
        
        // Calculate total price
        double totalPrice = 0;
        for (Book book : library) {
            totalPrice += book.price;
        }
        System.out.println("Total Price: $" + totalPrice);
    }
}
```

## üíª Example 8: Class with Utility Methods

```java
class Rectangle {
    double length;
    double width;
    
    void setDimensions(double length, double width) {
        this.length = length;
        this.width = width;
    }
    
    double calculateArea() {
        return length * width;
    }
    
    double calculatePerimeter() {
        return 2 * (length + width);
    }
    
    boolean isSquare() {
        return length == width;
    }
    
    void displayInfo() {
        System.out.println("Length: " + length);
        System.out.println("Width: " + width);
        System.out.println("Area: " + calculateArea());
        System.out.println("Perimeter: " + calculatePerimeter());
        System.out.println("Is Square? " + isSquare());
    }
    
    // Compare with another rectangle
    boolean isLargerThan(Rectangle other) {
        return this.calculateArea() > other.calculateArea();
    }
}

public class UtilityMethodsDemo {
    public static void main(String[] args) {
        Rectangle rect1 = new Rectangle();
        rect1.setDimensions(10, 5);
        
        Rectangle rect2 = new Rectangle();
        rect2.setDimensions(7, 7);
        
        System.out.println("=== Rectangle 1 ===");
        rect1.displayInfo();
        
        System.out.println("\n=== Rectangle 2 ===");
        rect2.displayInfo();
        
        // Compare rectangles
        System.out.println("\nIs rect1 larger than rect2? " + rect1.isLargerThan(rect2));
        System.out.println("Is rect2 larger than rect1? " + rect2.isLargerThan(rect1));
    }
}
```

## üìä Object Creation Steps

| Step | Description | Syntax |
|------|-------------|--------|
| **1. Declaration** | Declare reference variable | `Student s1;` |
| **2. Instantiation** | Create object using `new` | `new Student();` |

## üí° Best Practices

1. ‚úÖ **Use meaningful class names**
   ```java
   // Good
   class BankAccount { }
   
   // Bad
   class BA { }
   ```

2. ‚úÖ **Follow naming conventions**
   ```java
   // Class: PascalCase
   class StudentRecord { }
   
   // Variables/methods: camelCase
   String firstName;
   void calculateTotal() { }
   ```

3. ‚úÖ **Keep instance variables private (Encapsulation)**
   ```java
   class Student {
       private String name;  // Better than public
       public String getName() { return name; }
   }
   ```

4. ‚úÖ **Use 'this' for clarity**
   ```java
   void setName(String name) {
       this.name = name;  // Clear distinction
   }
   ```

5. ‚úÖ **Initialize objects properly**
   ```java
   Student s = new Student();
   s.name = "Alice";  // Initialize before use
   ```

## ‚ö†Ô∏è Common Pitfalls

```java
public class CommonPitfalls {
    public static void main(String[] args) {
        // 1. NullPointerException
        Student s1;
        // s1.name = "Alice";  // ERROR: s1 is null!
        s1 = new Student();  // Must initialize first
        s1.name = "Alice";   // Now OK
        
        // 2. Confusing reference assignment with object creation
        Student s2 = new Student();
        s2.name = "Bob";
        
        Student s3 = s2;  // NOT creating new object!
        s3.name = "Charlie";
        System.out.println("s2 name: " + s2.name);  // "Charlie" (same object!)
        
        // 3. Not using 'this' when parameter names match
        class Person {
            String name;
            
            void setName(String name) {
                name = name;  // BAD! Assigns parameter to itself
                // this.name = name;  // GOOD!
            }
        }
        
        // 4. Modifying one object affects reference copies
        Point p1 = new Point();
        p1.x = 10;
        Point p2 = p1;  // Same object
        p2.x = 20;
        System.out.println("p1.x: " + p1.x);  // 20 (affected!)
    }
}

class Student {
    String name;
}

class Point {
    int x, y;
}
```
