# üîó Creating Custom LinkedList

## üìñ Overview

Creating a **custom LinkedList** demonstrates doubly linked list implementation. Key concepts:
- **Node structure** - Data, previous, next pointers
- **Head/tail pointers** - Quick access to ends
- **Bidirectional** - Traverse forward and backward
- **Dynamic** - No fixed size, grows as needed

```java
public class CustomLinkedList<E> {
    private Node<E> head;
    private Node<E> tail;
    private int size;
    
    private static class Node<E> {
        E data;
        Node<E> prev;
        Node<E> next;
    }
}
```

---

## ‚≠ê Key Components

1. **Node class** - Stores data and links
2. **Head/tail pointers** - Track first/last nodes
3. **Add operations** - O(1) at ends, O(n) at index
4. **Remove operations** - O(1) at ends, O(n) at index
5. **Traversal** - Forward from head, backward from tail

## üíª Basic Implementation

```java
public class CustomLinkedList<E> {
    
    // Inner node class
    private static class Node<E> {
        E data;
        Node<E> prev;
        Node<E> next;
        
        Node(E data) {
            this.data = data;
        }
    }
    
    private Node<E> head;
    private Node<E> tail;
    private int size;
    
    // Constructor
    public CustomLinkedList() {
        this.head = null;
        this.tail = null;
        this.size = 0;
    }
    
    // Add at end
    public void add(E element) {
        Node<E> newNode = new Node<>(element);
        
        if (isEmpty()) {
            head = tail = newNode;
        } else {
            tail.next = newNode;
            newNode.prev = tail;
            tail = newNode;
        }
        size++;
    }
    
    // Add at beginning
    public void addFirst(E element) {
        Node<E> newNode = new Node<>(element);
        
        if (isEmpty()) {
            head = tail = newNode;
        } else {
            newNode.next = head;
            head.prev = newNode;
            head = newNode;
        }
        size++;
    }
    
    // Add at end (alias)
    public void addLast(E element) {
        add(element);
    }
    
    // Add at index
    public void add(int index, E element) {
        checkIndexForAdd(index);
        
        if (index == 0) {
            addFirst(element);
        } else if (index == size) {
            addLast(element);
        } else {
            Node<E> current = getNode(index);
            Node<E> newNode = new Node<>(element);
            
            newNode.prev = current.prev;
            newNode.next = current;
            current.prev.next = newNode;
            current.prev = newNode;
            
            size++;
        }
    }
    
    // Get element at index
    public E get(int index) {
        checkIndex(index);
        return getNode(index).data;
    }
    
    // Get first element
    public E getFirst() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        return head.data;
    }
    
    // Get last element
    public E getLast() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        return tail.data;
    }
    
    // Set element at index
    public E set(int index, E element) {
        checkIndex(index);
        Node<E> node = getNode(index);
        E oldData = node.data;
        node.data = element;
        return oldData;
    }
    
    // Remove at index
    public E remove(int index) {
        checkIndex(index);
        Node<E> node = getNode(index);
        return removeNode(node);
    }
    
    // Remove first
    public E removeFirst() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        
        E data = head.data;
        
        if (head == tail) {
            head = tail = null;
        } else {
            head = head.next;
            head.prev = null;
        }
        
        size--;
        return data;
    }
    
    // Remove last
    public E removeLast() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        
        E data = tail.data;
        
        if (head == tail) {
            head = tail = null;
        } else {
            tail = tail.prev;
            tail.next = null;
        }
        
        size--;
        return data;
    }
    
    // Remove specific element
    public boolean remove(Object obj) {
        Node<E> current = head;
        
        while (current != null) {
            if ((obj == null && current.data == null) ||
                (obj != null && obj.equals(current.data))) {
                removeNode(current);
                return true;
            }
            current = current.next;
        }
        
        return false;
    }
    
    // Helper: Remove node
    private E removeNode(Node<E> node) {
        E data = node.data;
        
        if (node == head) {
            return removeFirst();
        } else if (node == tail) {
            return removeLast();
        } else {
            node.prev.next = node.next;
            node.next.prev = node.prev;
            size--;
        }
        
        return data;
    }
    
    // Helper: Get node at index
    private Node<E> getNode(int index) {
        Node<E> current;
        
        // Optimize: Start from head or tail
        if (index < size / 2) {
            current = head;
            for (int i = 0; i < index; i++) {
                current = current.next;
            }
        } else {
            current = tail;
            for (int i = size - 1; i > index; i--) {
                current = current.prev;
            }
        }
        
        return current;
    }
    
    // Contains element
    public boolean contains(Object obj) {
        return indexOf(obj) != -1;
    }
    
    // Index of element
    public int indexOf(Object obj) {
        Node<E> current = head;
        int index = 0;
        
        while (current != null) {
            if ((obj == null && current.data == null) ||
                (obj != null && obj.equals(current.data))) {
                return index;
            }
            current = current.next;
            index++;
        }
        
        return -1;
    }
    
    // Last index of element
    public int lastIndexOf(Object obj) {
        Node<E> current = tail;
        int index = size - 1;
        
        while (current != null) {
            if ((obj == null && current.data == null) ||
                (obj != null && obj.equals(current.data))) {
                return index;
            }
            current = current.prev;
            index--;
        }
        
        return -1;
    }
    
    // Size
    public int size() {
        return size;
    }
    
    // Is empty
    public boolean isEmpty() {
        return size == 0;
    }
    
    // Clear
    public void clear() {
        Node<E> current = head;
        
        while (current != null) {
            Node<E> next = current.next;
            current.data = null;
            current.prev = null;
            current.next = null;
            current = next;
        }
        
        head = tail = null;
        size = 0;
    }
    
    // Index validation
    private void checkIndex(int index) {
        if (index < 0 || index >= size) {
            throw new IndexOutOfBoundsException("Index: " + index + ", Size: " + size);
        }
    }
    
    private void checkIndexForAdd(int index) {
        if (index < 0 || index > size) {
            throw new IndexOutOfBoundsException("Index: " + index + ", Size: " + size);
        }
    }
    
    // To array
    @SuppressWarnings("unchecked")
    public Object[] toArray() {
        Object[] array = new Object[size];
        Node<E> current = head;
        int i = 0;
        
        while (current != null) {
            array[i++] = current.data;
            current = current.next;
        }
        
        return array;
    }
    
    @Override
    public String toString() {
        if (isEmpty()) {
            return "[]";
        }
        
        StringBuilder sb = new StringBuilder("[");
        Node<E> current = head;
        
        while (current != null) {
            sb.append(current.data);
            if (current.next != null) {
                sb.append(", ");
            }
            current = current.next;
        }
        
        sb.append("]");
        return sb.toString();
    }
}
```

## üéØ Usage Example

```java
public class Main {
    public static void main(String[] args) {
        CustomLinkedList<String> list = new CustomLinkedList<>();
        
        // Add elements
        list.add("Alice");
        list.add("Bob");
        list.add("Charlie");
        System.out.println(list);  // [Alice, Bob, Charlie]
        
        // Add at beginning
        list.addFirst("Start");
        System.out.println(list);  // [Start, Alice, Bob, Charlie]
        
        // Add at index
        list.add(2, "Middle");
        System.out.println(list);  // [Start, Alice, Middle, Bob, Charlie]
        
        // Get elements
        System.out.println("First: " + list.getFirst());  // Start
        System.out.println("Last: " + list.getLast());    // Charlie
        System.out.println("At 2: " + list.get(2));       // Middle
        
        // Remove elements
        list.removeFirst();
        System.out.println("After removeFirst: " + list);
        
        list.removeLast();
        System.out.println("After removeLast: " + list);
        
        list.remove(1);
        System.out.println("After remove(1): " + list);
        
        // Contains
        System.out.println("Contains Alice: " + list.contains("Alice"));
        
        // Size
        System.out.println("Size: " + list.size());
    }
}
```

## üöÄ Advanced Features

### 1. Iterator
```java
public Iterator<E> iterator() {
    return new Iterator<E>() {
        private Node<E> current = head;
        private Node<E> lastReturned = null;
        
        @Override
        public boolean hasNext() {
            return current != null;
        }
        
        @Override
        public E next() {
            if (!hasNext()) {
                throw new NoSuchElementException();
            }
            lastReturned = current;
            E data = current.data;
            current = current.next;
            return data;
        }
        
        @Override
        public void remove() {
            if (lastReturned == null) {
                throw new IllegalStateException();
            }
            removeNode(lastReturned);
            lastReturned = null;
        }
    };
}
```

### 2. Reverse Iterator
```java
public Iterator<E> descendingIterator() {
    return new Iterator<E>() {
        private Node<E> current = tail;
        
        @Override
        public boolean hasNext() {
            return current != null;
        }
        
        @Override
        public E next() {
            if (!hasNext()) {
                throw new NoSuchElementException();
            }
            E data = current.data;
            current = current.prev;
            return data;
        }
    };
}
```

### 3. Reverse List
```java
public void reverse() {
    Node<E> current = head;
    Node<E> temp = null;
    
    while (current != null) {
        // Swap prev and next
        temp = current.prev;
        current.prev = current.next;
        current.next = temp;
        current = current.prev;  // Move to next (now prev)
    }
    
    // Swap head and tail
    if (temp != null) {
        temp = head;
        head = tail;
        tail = temp;
    }
}
```

### 4. Get Middle Element
```java
public E getMiddle() {
    if (isEmpty()) {
        throw new NoSuchElementException("List is empty");
    }
    
    Node<E> slow = head;
    Node<E> fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    
    return slow.data;
}
```

## ‚ö†Ô∏è Common Pitfalls

1. **Not updating both links**
```java
// ‚ùå Only update next pointer
newNode.next = head;
head = newNode;  // prev not set!

// ‚úÖ Update both
newNode.next = head;
head.prev = newNode;
head = newNode;
```

2. **Forgetting edge cases**
```java
// ‚ùå NPE on empty list
public E removeFirst() {
    E data = head.data;  // NPE if head is null
    head = head.next;
    return data;
}

// ‚úÖ Check empty
public E removeFirst() {
    if (isEmpty()) {
        throw new NoSuchElementException("List is empty");
    }
    // ... remove logic
}
```

3. **Not clearing references**
```java
// ‚ùå Memory leak
public void clear() {
    head = tail = null;
    size = 0;  // Nodes still linked
}

// ‚úÖ Clear all links
public void clear() {
    Node<E> current = head;
    while (current != null) {
        Node<E> next = current.next;
        current.data = null;
        current.prev = null;
        current.next = null;
        current = next;
    }
    head = tail = null;
    size = 0;
}
```

4. **Wrong boundary checks**
```java
// ‚ùå Allow index == size for add
private void checkIndex(int index) {
    if (index < 0 || index >= size) {
        throw new IndexOutOfBoundsException();
    }
}

// ‚úÖ Separate checks

    if (index < 0 || index >= size) {
        throw new IndexOutOfBoundsException();
    }
}

private void checkIndexForAdd(int index) {
    if (index < 0 || index > size) {  // Allow index == size
        throw new IndexOutOfBoundsException();

}
```

## üí° Best Practices

1. **Optimize traversal**
```java
// ‚úÖ Start from closer end
private Node<E> getNode(int index) {
    if (index < size / 2) {
        // Start from head
        Node<E> current = head;
        for (int i = 0; i < index; i++) {
            current = current.next;
        }
        return current;
    } else {
        // Start from tail
        Node<E> current = tail;
        for (int i = size - 1; i > index; i--) {
            current = current.prev;
        }
        return current;
    }
}
```

2. **Use sentinel nodes**
```java
// Dummy head and tail to simplify edge cases
```

## üìä Performance Analysis

| Operation | LinkedList | ArrayList | Notes |
|-----------|-----------|-----------|-------|
| **add()** | O(1) | O(1) amortized | LinkedList constant |
| **add(0, e)** | O(1) | O(n) | LinkedList wins at start |
| **add(i, e)** | O(n) | O(n) | Both need traversal |
| **get(i)** | O(n) | O(1) | ArrayList wins |
| **remove(0)** | O(1) | O(n) | LinkedList wins at start |
| **remove(i)** | O(n) | O(n) | Both need traversal |
| **contains()** | O(n) | O(n) | Both linear search |
| **iterator.remove()** | O(1) | O(n) | LinkedList better |

**When to use LinkedList:**
- Frequent insertions/deletions at ends
- Iterator-based modifications
- Don't need random access

**When to use ArrayList:**
- Random access needed
- More reads than writes
- Memory efficiency important

## üìö Interview Questions

**Q1: LinkedList vs ArrayList?**
A: LinkedList: O(1) add/remove at ends, O(n) random access. ArrayList: O(1) random access, O(n) insert/remove (shift elements).

**Q2: Singly vs Doubly linked list?**
A: Singly: One direction, less memory. Doubly: Both directions, easier deletion, more memory.

**Q3: Why keep tail pointer?**
A: O(1) addLast and removeLast. Without it, O(n) to traverse to end.

**Q4: How to detect cycle?**
A: Floyd's algorithm - slow/fast pointers. If they meet, cycle exists.

**Q5: How to find middle element?**
A: Two pointers - slow (1 step), fast (2 steps). When fast reaches end, slow is at middle.

**Q6: Time complexity of get(i)?**
A: O(n) - Must traverse i nodes. Can optimize by starting from closer end (head or tail).

**Q7: LinkedList as Queue?**
A: Perfect - O(1) addLast (enqueue) and removeFirst (dequeue).

**Q8: Memory overhead?**
A: Higher than ArrayList - Each node has 2 extra references (prev, next).

## üìö Summary

- **Doubly linked**: Each node has prev and next pointers
- **Head/tail**: Quick access to both ends (O(1))
- **Add/remove at ends**: O(1) with head/tail pointers
- **Add/remove at index**: O(n) - must traverse
- **Get/set**: O(n) - no random access
- **Use cases**: Frequent insertions/deletions at ends, implement Queue/Deque
- **Tradeoffs**: Better insert/delete, worse random access than ArrayList
- **Memory**: Higher overhead (2 pointers per node)
